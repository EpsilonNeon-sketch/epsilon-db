Use Epsilon DB in your projects

Quick Start

Include the library in your HTML file. It requires Supabase JS to be loaded first.

<!-- Load Supabase first --> <script src="https://cdn.jsdelivr.net/npm/@supabase/supabase-js@2"></script> <!-- Then Epsilon DB --> <script src="epsilondbs.js"></script>

Initialize

const db = new EpsilonDB({ apiKey: 'epk_live_...' });

Create a Collection

await db.createCollection('users');

Add a Document

const doc = await db.collection('users').add({ name: 'John', email: 'john@test.com' }); // doc._id = auto-generated ID

Get All Documents

const docs = await db.collection('users').get();

Get by ID

const doc = await db.collection('users').getById('abc123');

Update a Document

await db.collection('users').update('abc123', { name: 'Jane' });

Delete a Document

await db.collection('users').delete('abc123');

Delete a Collection

await db.deleteCollection('users');

List Collections

const cols = await db.listCollections();
